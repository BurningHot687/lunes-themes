# Contributing to Lunes Themes

Thank you for considering contributing to the Lunes theme family! This project is a calm, middle-ground workspace designed to blend the strengths of **Atom One Dark** and **Lunar Blue**, built specifically for Zed's flat architecture. Because of that, there are a few specific factors and regulations you must follow:

## Core Design Rules

1. **Strictly Borderless:** We are completely averse to borders, grid lines, or transparent/glassy backgrounds. Separate panels *only* using subtle background value steps.
2. **Simple Naming:** Use **Lunes** for dark themes and **Solas** for light themes. Keep color variants simple (e.g., `Lunes Indigo`, `Lunes Green`, `Solas Blue-Green`).
3. **Mid-Vibrancy Tokens:** Keep syntax colors readable but restrained. Don't be overly bright or overly dark.
4. **Simple Family:** Do not introduce more complex variations other than the basic color variants or Lunes/Solas variants. If you want a transparent or glassy theme, or a high contrast, accessible theme, please consider forking and making a new theme family.

Please ensure any pull requests reference our lineage to **One Dark** and **Lunar Blue** in their documentation!
